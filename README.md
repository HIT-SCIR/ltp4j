# 简介

ltp4j是语言技术平台[(Language Technology Platform, LTP)](https://github.com/HIT-SCIR/ltp)接口的一个Java封装。
本项目旨在使Java用户可以本地调用LTP。
本项目仍在开发之中，欢迎反馈bug。
